### How to run

#### clone the project: https://github.com/gurkanucar/address-book-project.git

```bash
  git clone https://github.com/gurkanucar/address-book-project.git
  cd address-book-project
```

 #### install the packages

```bash
  npm install
```

#### run
    
```bash
  npm start
```
